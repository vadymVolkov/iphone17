# iPhone 17 - Лендинг страница

Современная и адаптивная лендинг страница для продажи iPhone 17 с интерактивными элементами и профессиональным дизайном.

## 🚀 Особенности

- **Адаптивный дизайн** - отлично выглядит на всех устройствах
- **Современный UI/UX** - чистый и элегантный интерфейс
- **Интерактивные элементы** - анимации, эффекты наведения, модальные окна
- **Оптимизированная производительность** - быстрая загрузка и плавные анимации
- **SEO-оптимизация** - семантическая разметка и мета-теги
- **Доступность** - поддержка скрин-ридеров и клавиатурной навигации

## 📱 Секции сайта

1. **Hero Section** - Главный экран с призывом к действию
2. **Features** - Ключевые возможности iPhone 17
3. **Specifications** - Технические характеристики с интерактивными элементами
4. **Gallery** - Галерея изображений устройства
5. **Pricing** - Тарифные планы и модели
6. **Contact** - Форма обратной связи и контактная информация

## 🛠 Технологии

- **HTML5** - семантическая разметка
- **CSS3** - современные стили с CSS Grid и Flexbox
- **JavaScript (ES6+)** - интерактивность и анимации
- **Font Awesome** - иконки
- **Google Fonts** - типографика (Inter)

## 🚀 Быстрый старт

### Установка зависимостей
```bash
npm install
```

### Запуск локального сервера
```bash
npm start
```
или
```bash
npm run dev
```

### Альтернативный способ (Python)
```bash
npm run serve
```

Сайт будет доступен по адресу: `http://localhost:3000` или `http://localhost:8000`

## 📝 Доступные команды

- `npm start` - Запуск live-server на порту 3000
- `npm run dev` - Запуск с автоперезагрузкой при изменениях
- `npm run build` - Сборка проекта (статический сайт)
- `npm run lint` - Проверка кода на ошибки
- `npm run format` - Форматирование кода
- `npm run serve` - Запуск Python HTTP сервера

## 🎨 Дизайн-система

### Цветовая палитра
- **Primary**: #007AFF (синий Apple)
- **Secondary**: #5856D6 (фиолетовый)
- **Accent**: #FF3B30 (красный)
- **Text Primary**: #1D1D1F
- **Text Secondary**: #86868B
- **Background**: #FFFFFF / #F5F5F7

### Типографика
- **Шрифт**: Inter (Google Fonts)
- **Заголовки**: 600-700 font-weight
- **Основной текст**: 400 font-weight

### Адаптивность
- **Desktop**: 1200px+
- **Tablet**: 768px - 1199px
- **Mobile**: до 767px

## 🔧 Кастомизация

### Изменение цветов
Отредактируйте CSS переменные в файле `css/style.css`:
```css
:root {
    --primary-color: #007AFF;
    --secondary-color: #5856D6;
    /* ... другие переменные */
}
```

### Добавление изображений
Поместите изображения в папку `images/` и обновите пути в HTML.

### Модификация контента
Отредактируйте текст и структуру в файле `index.html`.

## 🚀 Развертывание

### GitHub Pages
1. Загрузите код в GitHub репозиторий
2. Включите GitHub Pages в настройках репозитория

2. Настройки сборки: Build command: `npm run build`

1. Импортируйте проект в Vercel
2. Настройки по умолчанию подойдут для статического сайта
